APA POOL STRATEGY

This script will help to build a strategy of possible combinations of players for APA Pool team
1. Update Team and skill level of the players in team_9ball and team_8ball 
2. Run the script
3. Press 8 or 9 to select pool format from the main menu
4. Press number to select player
5. Press q to save and return to main menu
6. Press r to change previously selected player
